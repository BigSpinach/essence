## JSON

> json 是一种数据类型格式
>
> json对象    属性名必须用双引号包起来
>
> ```javascript
> var obj = {"a":1,"b":2};
> var ary = [{"a":1,"b":2},{"a":12,"b":22},{"a":13,"b":24}];
> ```
>
> json字符串	将json对象用 `‘’` 包起来
>
> ```javascript
> var str1 = '{"a":1,"b":2}';
> var str2 = '[{"a":1,"b":2},{"a":12,"b":22},{"a":13,"b":24}]';
> ```
